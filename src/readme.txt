Для того чтобы приложение начало работать необходимо:
1. Заполнить данные в файле congig.php
	a. Указать хост базы данных(БД)				  	// define( 'DB_HOST', 'localhost' );
	б. Указать имя пользователя для БД   			// define( 'DB_USER', 'root' );
	в. Указать пароль пользователя для доступа к БД // define( 'DB_PASSWORD', '' );
	г. Указать имя таблицы БД  						// define( 'DB_NAME', 'test_webdev_base' );
2. Необходимо указать данные для авторизации 
	а. Указать ID вашего приложентия FB 			// define( "FB_CLIENT_ID", "ID вашего приложентия" );
	б. Указать секретный ключ вашего приложентия FB // define( "FB_SECRET", "секретный ключ" );
	в. Указать имя вашего сайта 					// define ( "FB_REDIRECT", "имя вашего сайта" );

В папке /dump_db находится дамп базы данных с демо данными, вы можете использовать его для востановления таблиц или установить значение true для константы "DEMO_UPLOAD" в файле congig.php тогда при первом запуске в таблице будет сгенерировано ряд записей для демонстрации работы приложения.

В папке /lib -используемые библиотеки.

В папке /app -файлы приложения.
